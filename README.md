# ds-guide
(WIP) Calls out mechanics and boss skills.

## Commands
### `ds [off/on]'
- Toggle off/on
- Default is on
- Arguments are optional

### `ds [stream]'
- Toggle stream mode (Disable notices. Send chat messages quietly to yourself)
- Default is off
